Adebug: a 68000/68008/68010/68012/68020/68030/68851/68881/68882 DEBUGGER
                for ATARI ST, MEGA ST, STACY ,STE and TT
			written by Brainstorm
		formerly distributed by Arobace (France)

	Adebug Reloaded v2.13 is now released under the GPL licence

Key features:
-------------

* Works in high, medium, low resolution on all memory configurations.

* Gets command line:
	For relocatable files:
		<filename> [command string]
	For binary files:
		-b <filename>

* Recognizes DRI debug symbols format (8 characters), GST & extended 
  HISOFT format (22 characters)

* Has a command line history
